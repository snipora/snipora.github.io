# ::rocket:: Getting Started

Welcome to ::snipora:: Snipora! This guide will help you get up and running in ::timer:: minutes.

## ::sticky-note:: What is a Snippet?

A snippet is a small piece of text that you can save and quickly reuse. Instead of typing the same text over and over, just store it as a snippet and insert it instantly whenever you need it.

Think of it as a clipboard that remembers everything you save.

**Examples:**
- ::bot:: AI prompts [::dot:: instructions, personas, writing styles]{.note}
- ::reply-all:: Email replies or support answers [::dot:: FAQ, canned replies]{.note}
- ::at-sign:: Email signatures [::dot:: professional, casual, contact info]{.note}
- ::code-xml:: Code blocks [::dot:: files templates, build scripts, utility types]{.note}
- ::square-terminal:: Command-line commands [::dot:: git commands, docker runs, build scripts]{.note}

## ::hard-drive-download:: Installing Snipora

1. Download ::snipora:: Snipora from the [::download:: Download](download.md) page
2. Install ::snipora:: Snipora:
   - ::social/windows:: **Windows**: Run the `.exe` installer
   - ::social/apple:: **macOS**: Open the `.dmg` and drag Snipora to Applications
   - ::social/linux:: **Linux**: Make the AppImage executable (`chmod +x`) and run it
3. Launch ::snipora:: Snipora

> [!TIP]
> To make Snipora always available after your PC starts, enable `Autostart` in the Settings.

## ::app-window:: Opening the Popup

Press the global ::command:: shortcut to open Snipora from anywhere:

::social/windows:: **Windows** / ::social/linux:: **Linux** ::dot:: <kbd>Ctrl</kbd>::plus::<kbd>Shift</kbd>::plus::<kbd>Space</kbd>

::social/apple:: **macOS** ::dot:: <kbd>Cmd</kbd>::plus::<kbd>Shift</kbd>::plus::<kbd>Space</kbd>

A small ::app-window-mac:: popup window will appear in the center of your ::monitor:: screen.

## ::sticky-note-plus:: Creating Your First Snippet

1. Open the ::app-window:: main-window where you can manage your snippets
2. Click the [::plus:: New Snippet] button [or press <kbd>Ctrl</kbd>::plus::<kbd>N</kbd>]{.note}
3. Enter a ::case-lower:: label for your snippet into the text input
4. Then enter or paste your ::text-align-start:: snippet content in the text area
5. Add ::tags:: tags to organize your snippet [(optional but recommended)]{.note}
6. Click the [::save:: Save Snippet] button [or press <kbd>Ctrl</kbd>::plus::<kbd>S</kbd>]{.note} to save

That's it! Your snippet is now saved and ready to use.

## ::clipboard-paste:: Inserting a Snippet

1. Open the ::app-window-mac:: popup with <kbd>Ctrl</kbd>::plus::<kbd>Shift</kbd>::plus::<kbd>Space</kbd>
2. Start typing to ::search:: search, or navigate with ::arrow-up-down:: arrow keys
3. Press ::corner-down-left:: <kbd>Enter</kbd> to insert the selected snippet directly into your active application

The ::app-window-mac:: popup will close automatically after inserting.

## ::tags:: Understanding Tags

Snipora uses ::tags:: **tags** instead of folders to organize your snippets. Here's why:

### ::circle-question-mark:: Why Tags?

- ::split:: **Flexibility**: A snippet can have multiple tags (e.g., "work", "email", "important")
- ::clock-1:: **Fast filtering**: Click a tag to instantly filter all snippets with that tag
- ::folder-tree:: **No hierarchy**: Flat structure means no deep folder navigation
- ::search:: **Easy to find**: Search across all tags instantly

### ::circle-pile:: Managing Tags

- ::plus:: **Create tags**: Type a new tag name in the tags input when creating/editing a snippet
- ::x:: **Remove tags**: Click the ::x:: on a tag to remove it
- ::list-filter:: **Filter by tag**: Click any tag in the sidebar to see only snippets with that tag
- ::list:: **View all tags**: See all your tags in the sidebar with snippet counts

## ::presentation:: Quick Workflow Demo

Let's say you frequently send the same email signature:

1. Open the main ::app-window:: window
2. Create a new snippet [(<kbd>Ctrl</kbd>::plus::<kbd>N</kbd>)]{.note}
3. Paste your signature into the content area
4. Add tags: `email`, `signature`
5. Save [(<kbd>Ctrl</kbd>::plus::<kbd>S</kbd>)]{.note}

Now whenever you need it, just:
1. Open the ::app-window-mac:: popup [::dot:: (<kbd>Ctrl</kbd>::plus::<kbd>Shift</kbd>::plus::<kbd>Space</kbd>)]{.note}
2. Type `"signature"` to ::search:: search
3. Press <kbd>Enter</kbd> to copy the snippet into your ::clipboard:: clipboard
